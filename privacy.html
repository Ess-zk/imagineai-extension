Política de Privacidade – ImagineAI

A extensão ImagineAI – Gerador de Imagens respeita a privacidade dos usuários.

Esta extensão não coleta, armazena ou compartilha dados pessoais dos usuários.

As descrições de texto fornecidas pelo usuário são utilizadas exclusivamente para a geração das imagens solicitadas e não são armazenadas para qualquer outro fim.

As configurações da extensão são armazenadas localmente no navegador do usuário por meio da API chrome.storage, sem qualquer transmissão para servidores externos.

A extensão não utiliza cookies, não realiza rastreamento e não emprega ferramentas de análise ou publicidade.

Em caso de dúvidas, o contato deve ser feito pelo e-mail informado na página da extensão na Chrome Web Store.
